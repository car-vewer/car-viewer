# Cars Viewer

![Banner](/car-viewer/static/banner.png)

## Introduction

Cars Viewer is a modern web application built with **Go**, **HTML**, and
**CSS** that lets users browse, search, filter, explore, and compare detailed
information about different cars.

---

## Overview

-   Browse all available cars
-   Search by car name
-   Advanced filtering
-   Compare cars
-   View detailed specifications


![Home](/car-viewer/static/home.png)


## Usage

### 1. Run the API

Inside the `api/` directory:

``` bash
make run
```

or

``` bash
npm install
npm start
```

### 2. Run the web application

From the project root:

``` bash
go run .
```

Then open:

``` text
http://localhost:8080
```

- Full-stack Go + Node.js web app
